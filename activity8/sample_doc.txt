Introduction to Vector Databases

A vector database is a specialized database designed to store and query high-dimensional vectors efficiently. These vectors are typically generated by machine learning models and represent semantic meaning of text, images, or audio.

Unlike traditional databases that match exact values, vector databases perform similarity search. This means they can find records that are conceptually similar to a query, even if they share no exact keywords.

Qdrant is an open-source vector database written in Rust. It supports cosine similarity, dot product, and Euclidean distance metrics. Qdrant can run as a standalone Docker container or in-memory for development purposes.

What is RAG?

Retrieval-Augmented Generation (RAG) is a technique that enhances large language models by giving them access to external knowledge at inference time. Instead of relying solely on what the model learned during training, RAG first retrieves relevant documents from a knowledge base, then passes those documents as context to the model.

The key steps in a RAG pipeline are:
1. Ingestion: Documents are split into chunks, embedded into vectors, and stored in a vector database.
2. Retrieval: When a user asks a question, the question is embedded and the most similar document chunks are fetched.
3. Generation: The retrieved chunks are added to the prompt, and the language model generates an answer grounded in that context.

RAG reduces hallucinations because the model is given factual source material to reference. It also allows the knowledge base to be updated without retraining the model.

Chunking Strategies

Chunking is the process of dividing a document into smaller pieces before embedding. The right chunk size depends on the use case.

Fixed-size chunking splits text by a set number of characters or tokens, with optional overlap to preserve context across boundaries. It is simple and predictable but may cut sentences mid-thought.

Semantic chunking uses an embedding model to detect topic shifts and splits text only when the meaning changes significantly. This produces more coherent chunks but is slower and more expensive.

Overlap is a technique where consecutive chunks share some characters or tokens at their boundaries. This helps ensure that ideas spanning a boundary are not lost.


JSON (JavaScript Object Notation)

JSON is a lightweight data interchange format that is easy for humans to read and write, and easy for machines to parse and generate. It is based on a subset of the JavaScript programming language but is language-independent, meaning it can be used with virtually any programming language.

JSON represents data as key-value pairs enclosed in curly braces. Keys must be strings wrapped in double quotes, and values can be strings, numbers, booleans, arrays, objects, or null. Arrays in JSON are ordered lists of values enclosed in square brackets.

A simple JSON object looks like this: {"name": "Alice", "age": 30, "active": true}. JSON is widely used in REST APIs to send and receive data between a client and a server. Most programming languages have built-in libraries to parse JSON strings into native data structures and serialize objects back into JSON strings.

Node.js

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside of a web browser. It was created by Ryan Dahl in 2009 and is built on Chrome's V8 JavaScript engine.

Node.js uses an event-driven, non-blocking I/O model, which makes it lightweight and efficient for building scalable network applications. Instead of creating a new thread for each request like traditional servers, Node.js handles many connections concurrently using a single-threaded event loop.

Node.js comes with npm, the Node Package Manager, which is the largest ecosystem of open-source libraries in the world. Developers use Node.js to build web servers, REST APIs, real-time chat applications, command-line tools, and microservices.

Popular frameworks built on top of Node.js include Express.js for web servers, Fastify for high-performance APIs, and Socket.io for real-time bidirectional communication.

Python

Python is a high-level, interpreted, general-purpose programming language known for its clean and readable syntax. It was created by Guido van Rossum and first released in 1991. Python emphasizes code readability and simplicity, which makes it one of the most popular programming languages for beginners and professionals alike.

Python supports multiple programming paradigms including procedural, object-oriented, and functional programming. It has a large standard library that covers everything from file I/O and networking to regular expressions and data serialization.

Python is widely used in web development with frameworks like Django and Flask, in data science and machine learning with libraries like NumPy, Pandas, and scikit-learn, and in automation and scripting. The language uses indentation to define code blocks instead of curly braces, which enforces a consistent and readable code style.

Python's package manager, pip, allows developers to install thousands of third-party libraries from the Python Package Index (PyPI). The language also supports virtual environments, which isolate project dependencies to avoid version conflicts between projects.


Java

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It was originally developed by James Gosling at Sun Microsystems and released in 1995. Java follows the principle of "write once, run anywhere," meaning compiled Java code can run on any platform that has a Java Virtual Machine (JVM) installed without needing to be recompiled.

Java is statically typed, which means variable types must be declared at compile time. This helps catch errors early and makes large codebases easier to maintain. Java uses garbage collection to automatically manage memory, freeing developers from manually allocating and deallocating memory.

Java is widely used in enterprise applications, Android mobile development, web backend services, and large-scale distributed systems. Popular frameworks include Spring Boot for building REST APIs and microservices, Hibernate for database object-relational mapping, and Maven or Gradle for project build management.

Java's strong ecosystem, backward compatibility, and performance make it one of the most used programming languages in the world, particularly in corporate and financial environments.

CSS (Cascading Style Sheets)

CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, spacing, and visual appearance of web pages. It separates content from design, allowing developers to change the look of an entire website by editing a single stylesheet.

CSS works by selecting HTML elements and applying style rules to them. Selectors can target elements by tag name, class, ID, attributes, or their relationship to other elements. Properties like color, font-size, margin, padding, and display control how elements appear on screen.

CSS introduced the box model, which describes how every HTML element is surrounded by padding, borders, and margins. Understanding the box model is fundamental to controlling layout in web development.

Modern CSS includes powerful layout systems such as Flexbox and CSS Grid. Flexbox is designed for one-dimensional layouts, aligning items in a row or column. CSS Grid is designed for two-dimensional layouts, allowing precise placement of elements in rows and columns simultaneously.

CSS also supports responsive design through media queries, which apply different styles depending on the screen size or device type. This allows a single website to look good on both desktop monitors and mobile phones.

HTML (HyperText Markup Language)

HTML is the standard markup language used to create the structure of web pages. It was created by Tim Berners-Lee in 1991 and is the backbone of every website on the internet. HTML describes the structure of a web page using a series of elements represented by tags.

HTML tags are keywords surrounded by angle brackets. Most tags come in pairs with an opening tag and a closing tag, such as paragraph tags that wrap text content. Some tags are self-closing, such as the image tag or the line break tag. Tags can have attributes that provide additional information, such as a link tag with an href attribute pointing to a URL.

HTML5 is the latest major version of HTML and introduced many new semantic elements such as header, footer, article, section, and nav. These semantic elements give meaning to the structure of a page, making it more accessible to screen readers and easier for search engines to understand.

HTML forms allow users to input data that can be submitted to a server. Form elements include text fields, checkboxes, radio buttons, dropdown menus, and submit buttons. HTML works together with CSS for styling and JavaScript for interactivity to form the foundation of modern web development.
